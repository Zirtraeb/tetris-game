# Tetris em Python
Projeto focado na criação de jogos clássicos utilizando a linguagem Python e a biblioteca Pygame.

# ⚒️Funcionalidades⚒️
- Movimentação e rotação de peças (Tetrominós): Controle completo das peças no eixo horizontal e sistema de rotação.
- Sistema de colisão: Verificação de limites das bordas do ecrã e de peças já fixadas.
- Detecção e limpeza de linhas: Identificação automática de linhas preenchidas, remoção e lançamento de novas peças.
- Sistema de pontuação: Atribuição de pontos por cada linha eliminada.
- Interface gráfica (Grid): Renderização visual do tabuleiro e das peças utilizando cores distintas.
- Lógica de próxima peça: Sistema de antecipação e geração aleatória da peça seguinte.
- Estado de fim de jogo: Deteção de quando as peças atingem o topo do ecrã e opção de reinício.
